# mikutter_tdr
## これなん
夢の国の情報を表示するデータソースを提供します．  
現在以下の情報に対応しています．
- パレード・ショーのスケジュール
- アトラクションの運営状況・待ち時間
- ファストパス発券状況
- レストラン情報
- 天気予報
